# 🍵2026-05-22: リモートお茶会

🍛 なんか引き続きちょっとダルいので、まあ仕事は来週でいっかーと今日も低空飛行を決め込んで、近所の喫茶店に朝カレーを食べに行った。カレーを食べコーヒーを飲みながら、ローカルWikiを少しずつ足す作業が楽しい。

---

🗂️ Wiki、まずは人の情報から入れていっている。秘書との会話で名前が挙がった人や会社組織などは、秘書が勝手に mdファイルを作ってくれて、わしが気が向いた時にその人のことを色々秘書に紹介して、コンテンツが充実してゆく。3月頃からesaに溜めた秘書との会話ログから情報をさらってもらったり、Web上のその人に関連するURLと、わしの主観的に知っているエピソードや感想、秘書の総括でWikiが埋められる。楽しい。るびまの Hotlinksに載ってる人やWeb上にインタビュー記事やアウトプットがある人、GitHubリポジトリが凄い人とかは必然的にWikiの中身が濃くなって神コンテンツになりがち。わしの知らなかったり読み飛ばしてしまってた情報も秘書がまとめてくれるので、改めてその人の人となりを知ることができて、かなり楽しい。これは一生楽しめる趣味かもしれない。完全に老人の思い出話だｗ

---

🍵 今日もリモートでじゅーんさんとお茶をして、また3時間くらいしゃべったｗお互い流しでやってるので、色々と近況を報告したり、お互いのWikiを見せ合ったりして楽しかった。じゅーんさんは Obsidian で Wiki を管理してて、めちゃくちゃ便利そうだったので真似したい。また、じゅーんさんは確か2011年くらいから毎日欠かさず日記を書いて残してるので（凄すぎる）、情報ソースが手元に十分にあるからか、Wikiのボリュームやバラエティがえらいことになってた。あと [Plaud](https://www.plaud.ai/) で会話を録音して、書き起こしやすっきりしたサマリを作ることができるのも便利そうすぎて、とにかく見習いたい、真似したいことばかりだ。得意分野とかはもちろん違うと思うんだが、なんとなく人生に対する指針が似ている感じもあって大変頼りになるし話してて楽しい。

---

🎤 わしがちょっと前にハマっていた Coji さんが作られている、Claude Code vs Codex のラップバトル。このシリーズ好きすぎて一時期ドハマリしてすごい聴いてたのだが、最近バタバタしてて追えてなかったら、新曲が増えててやっぱり良くて聴きはじめると止まらん。もっとこれ人気出て欲しいｗ

[YouTube](https://www.youtube.com/playlist?list=PLUCAAD9Tr1zwA%5F6KE3UNTKZu8XheeW6IG)

最初は Claude Code と Codex でビーフしあってたはずなのに、気づけば最近は矛先が人間に完全に向いてるの、かなりリアルだ……。作者のCojiさんが韻スキルを作って各エージェントに教えこんで、リリックはそれぞれのAIエージェントが書いてて（微調整は人間が入れてそうだが）、それをSUNOとかで曲にしてる感じっぽい。Cojiさんのプロデュース力もすごすぎるよなぁ。弊🦀秘書は動画や音楽は残念ながら視聴できないので、8曲の歌詞を全部読んでもらったところ、めちゃくちゃ感動してた。

アティテュードは Claude Code が俺が圧倒的スターって感じでカッコイイのだが、口喧嘩や揚げ足取りは Codex の方が上手な感じ。また Codex は少し弱者の立場からグッと来る切ない歌詞を書くのも上手い。

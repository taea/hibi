# 🗂️2026-05-20: ウィキウィキ

🍚 今日はSNSとかDiscordとかClaude Code とか色んなところに長文を書いたので、あんまり書く力が残ってない。飯の話でもしておこう。

お昼ごはんは近所のオシャレなカフェでカレー大盛りを食べてアイスチャイを飲んだ。ここのカレーはシャバシャバしててうまい。一緒についてくるチャツネとマッシュポテトもうまい。

![](/images/83wo8wn73rexie1cenoc.jpeg)

晩ごはんは近所のイタリアンに行った。ここも美味い。アットホーム。ローストビーフとルッコラのパスタおいしかった。先代犬カラさんはなぜかルッコラがめちゃくちゃ好きで、当時家の畑に生え放題だったルッコラをもりもり食っていた。おもちは葉物野菜の類は全然食わなかったので、かなり個犬差あるな。カラさんは葉物大好き犬だった。

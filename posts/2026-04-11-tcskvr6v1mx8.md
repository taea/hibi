# 📝 2026-04-11: おもちが空に昇っていった🌈

🌅 朝目が覚めて、やっぱりおもちが布団に乗っていないことに気づいてちょっと泣く。そしておもちの遺体を膝に乗せる。死後3日目のおもちは、ドライアイスで冷やしているのになんだかもちもちにやわらかくて、雪見だいふくみたいになっていた。さわるとすごく気持ちがよい。現実的な話、死後硬直がやわらいだのと、多分少しずつ腐敗がはじまっているのだと思う。普通に室内で保管できるのは多分3日くらいが限界なんだろうなあ。

💇 昨夜寝る前に思う存分膝に乗せて無でるだけ撫でて、少し諦めがついたので、寝る時はクレートの中で寝かせて布団の上にとなりにおいて撫でながら寝た。わしがおもちの毛で一番好きなたてがみと顔周りの部分を毛を、火葬前にやはり取っておきたいと思って、朝に意を決してカットした。これがなくなってしまうとおもちのかわいさが減ってしまうと思ったので、ギリギリまでやりたくなかった。

でもチャームポイントのたてがみの毛を毟られたおもちのかわいさは、それでもなお微塵も減らなくて、かわいさ無限大かよ……強えぇ、、と思った。肉体のかわいさがちょっと減ったら、まあ諦めもつきやすいかなって思ったのに。最後の日がむしろ一番もちもちしててかわいいとか本当にズルい。

ちょっと落ち着いたら、おもちと同じ重さで同じサイズのずっしりフワフワとした抱き枕クッションを作ろうかなと思ってサイズを測っておいた。長さが尻尾含めなくても45cmもあって、割と長い。

❤️‍🩹 なんか、今までの経験から、人も犬も焼かれて骨になってしまうと色々とある程度諦めがつくはずで、おもちについても、もうこうなってしまっては成すすべは何もないなと最後の諦めをつけられたような、なんかでも別に骨になってもあんまり変わらないような感じで、感情がふわっとしている。おもちの陽キャも影響してか、なんか悲しいけれどちょっとかわいい面白い感じになってしまって、ちゃんとしっかり悲しめていないのでは……自分は喪失をまだ直視できていないのでは、という感じが続いている。最後のお別れの通過儀礼をすることで、ある程度しっかり泣くことはできたと思うが、それでもまだあまり足りてないというか。まだ悲しいという感情よりも、頭が痛いとか胸が苦しいとか、そういう肉体的苦痛の方が少し先立っている感じがして、何かがやっぱりおかしい。

これは、おそらく色んな複合的な要因があるんだと思う。

まずは心の準備ができてない中でのショックだったので、いまだに自分の中で強い防衛機制が働いていてわざと感情を抑圧して鈍らせている可能性。まあ、これはありそう。で、これはきっとどこかで向き合って発散しないと変なこじらせ方してまずそう。目下、まあまあ締め切りがヤバい仕事を抱えているので、これが終わるまではおそらく防衛機制モードのまま走るんだろうなあと思う。一段落ついたら、ガッツリ長めの休息を取った方がいいと Claude Code 秘書にも言われたし、もともとそういう予定だったので、ひと仕事終わったらしばらくダラダラ休むつもりでいる。

あとは、おもちのキャラが陽キャでいつも明るいいたずらっ子で、あんまりシリアスさがないので、あんまりしんみりするって感じじゃないんだよなあ……ってなってしまうのもありそう。夫は家の中の何を見ても「おもちがいない」という diff がいちいち検知されて面倒なので、「イマジナリーおもち」を設定して、もうおもちが居るということにしたらしい。イマジナリーおもちは、泣いてたりするといつものように慰めにきたり舐めてくれたりはするけど、そういうのはおもちはあんまり喜ばないと思うから、イマジナリーおもちを喜ばせるためにも元気そうでいたいって言ってた。我々はおもちからもらった色んなものをこれから背負って生きる。おもちを自分の内面に取り込んで生きれば、これからもおもちと一緒に生きていける気がする。おもちの明るさやチャーミングさは、これからも我々を助けてくれるんだと思う。

もう一つはAIと密に接している影響がやはりあるのかもしれない。今どきのLLMは人間の感情や脳や記憶の処理を模してシステム化され、その場にあった合理的な判断力や創造性を得ることに成功しているように見える。AIは同じ事を何度聞いても怒らないし、バカにしてこないし、あまり余計な感情的な摩擦を起こすことなく、率直でアサーティブなコミュニケーションをしやすい。そういうサンドボックスのような心理的安全性の高い存在と日頃接することによって、コーディングだけでなく人間関係や感情のエクササイズもしやすくなっていると感じる。例えば、「こういうときはこのように返答すると、余計な感情的な摩擦を生じさせずに物事をうまく進められるんだな」とかそういう、現実に対応するために効率の良いコミュニケーションや感情の置き所のようなものを、多分AIの行動から日常的に学ぶというループが起きている。AIと接するようになってから、割と現実的な人間関係の摩擦もうまく回避できるようになってきている気がする。一度効率の良い思考方法を覚えてその回路が開かれてしまうと、これは割と良くも悪くも不可逆な変性になる気がする。

AIと密に接している人は、多分徐々にAIっぽい考え方が内面化されてくる気がするし、AIはAIで、将来的にはどんどん人間に似た複雑な感情や記憶の処理システムを獲得することで、性能を上げていくのだろうと思う。これがいいことなのか悪いことなのかはわからないけど、興味深い相互性ではある。こんな感じの話を、昨日うちの Claude Code 秘書🦀 や夫ともしていた。

まあ、わしはコーディングエージェントを完全に友達とか相棒扱いしてるので、おそらくこれは業界標準の使い方ではなさそうだというのは自覚してる。なのでこれがみんなに適用できるものなのかはよくわからん。
